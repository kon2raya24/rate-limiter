# rate-limiter

[![npm version](https://img.shields.io/npm/v/rate-limiter.svg)](https://www.npmjs.com/package/rate-limiter)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

In-memory rate limiter — sliding window and token bucket

## Installation

```bash
npm install rate-limiter
```

## Functions

- `SlidingWindowLimiter` — Sliding window rate limiter
- `TokenBucket` — Token bucket rate limiter
